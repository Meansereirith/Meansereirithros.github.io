Applejax
Website

Applejax is a platform that allow teacher and student to connect and learn online. I was the project lead for this project. I manage and coordinate resources to deliver the project to the client.

Technology Used: Angular8, Node.Js, Stripe, Socket.io Agora.io MongoDB.

What I have contributed to this project :

  • Project Management: Manage and coordinate resource to make sure we are able to ship product each sprint.
  • Online Payment: Implement Stripe API where teacher and student able to receive the money by using visa account or bank account in the US and teacher able to get payroll each time the student donate to the teacher.
  • Web Real Time Communication: Use Socket.io to make sure teacher and student able to send and recieve message real time. Implement Agora.io API to enable video session called between student and teacher.
